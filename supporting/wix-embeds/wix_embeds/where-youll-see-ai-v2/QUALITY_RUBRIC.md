# Quality Rubric - v2 Modular Embeds

## 1) Looks Smeal-native

- Uses Inter and matches page color temperature.
- Feels like a section of the site, not a detached widget.
- Avoids shiny gradients and gimmicky ornamentation.

## 2) Not AI-looking

- No excessive chips or icon clutter.
- No dashboard-style mini widgets.
- No over-animated transitions.

## 3) Readability

- Main point in each embed is clear in under 20 seconds.
- Scanning order is obvious without interaction.
- Reveal text adds value, not repetition.

## 4) Accessibility

- Visible keyboard focus on toggle buttons.
- Contrast is legible at body text sizes.
- `aria-expanded` state updates correctly.

## 5) Modular maintainability

- Each embed can be edited independently.
- Replacing one embed does not break spacing rhythm.
- Mobile stack remains coherent after content edits.

# Section Blueprint - Where You'll See AI (Accounting)

## Objective

Create a premium editorial section that feels native to the Smeal site and avoids "AI-looking" widget aesthetics.

## Component map

1. Section heading (native Wix text block, outside embeds).
2. Featured embed: Auditing (`01-featured-auditing.html`).
3. Support embed: Financial Reporting (`02-support-financial-reporting.html`).
4. Support embed: Tax & Compliance (`03-support-tax-compliance.html`).
5. Support embed: Internal Controls (`04-support-internal-controls.html`).
6. Full-width context embed with takeaways (`05-context-takeaways.html`).

## Design tokens

- `--smeal-navy: #0A2E63`
- `--smeal-blue: #0D4B95`
- `--smeal-accent: #1E6FD9`
- `--surface: #FFFFFF`
- `--surface-soft: #F8FBFF`
- `--border: #DBE7F7`
- `--text: #163F79`

## Typography

- Font: `Inter, "Segoe UI", Arial, sans-serif`
- Headline: strong, tight spacing, no decorative treatment
- Body: line-height `1.5-1.6`

## Interaction ceiling

- Allowed: single `Read more` toggle per embed
- Allowed: subtle focus state
- Not allowed: nested accordions, multi-step animation, ornamental transitions

# Code of Conduct

## Our Commitment
This project is a learning-focused collaboration space. We want students, contributors, and maintainers to participate in a respectful, inclusive, and constructive way.

## Expected Behavior
- Be respectful and collaborative.
- Assume good intent and ask clarifying questions when needed.
- Give actionable feedback tied to ideas, code, or docs.
- Keep discussions focused on improving the project and helping others learn.

## Unacceptable Behavior
- Harassment, hate speech, personal attacks, or intimidation.
- Doxxing or sharing private information without consent.
- Disruptive behavior that repeatedly derails collaboration.
- Sexualized language or unwelcome sexual attention.

## Scope
This applies to repository spaces (issues, pull requests, discussions, commits, comments) and project-related communication channels.

## Reporting
To report conduct issues, contact project maintainers through GitHub direct contact channels (maintainer profiles and repository contact routes).  
If there is immediate safety risk, prioritize personal safety and local emergency services first.

## Enforcement
Maintainers may take any action they determine appropriate, including warning, content removal, temporary restriction, or permanent ban from project spaces.

## Attribution
This document is informed by common open-source code of conduct practices, including Contributor Covenant principles.
